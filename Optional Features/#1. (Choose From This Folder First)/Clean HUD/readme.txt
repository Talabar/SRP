==============
=  MOD INFO  =
==============

Mod Name			: Clean HUD
Version				: 1.0
Author				: TehGlitch (modifications by Talabar)
Release Date			: 10.25.2014 (11.11.2020)

=================
=  DESCRIPTION  =
=================

This mod changes the HUD for a cleaner and more immersive experience.

NOTE: This version re-allocates the minimap to be in the top left corner (similar to vanilla) as opposed to the bottom left (default Clean HUD).

==================
=  INSTALLATION  =
==================

Extract files to your S.T.A.L.K.E.R. CS folder.

===================
=  COMPATIBILITY  =
===================

It should be compatible with any mod that don't change the HUD.
